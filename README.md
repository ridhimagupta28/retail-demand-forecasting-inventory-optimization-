# Retail Demand Forecasting & Inventory Optimization

An end-to-end data analytics and machine learning project using the Walmart M5 dataset to analyze sales trends, forecast demand, and generate inventory optimization insights.

## Tech Stack

- SQL (PostgreSQL)
- Python
- XGBoost
- ARIMA
- Power BI

## Project Workflow

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Business Insights using SQL
- Demand Forecasting
- Interactive Power BI Dashboard

## Dataset

- Walmart M5 Forecasting Dataset
